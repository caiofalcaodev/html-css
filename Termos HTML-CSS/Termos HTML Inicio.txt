Tags HTML:
-
Head: > cabeçario do site
body: > corpo = conteúdo do site
h1: > Tag para título grande
p: > Tag para paragrafo 
hr: > Tag para uma linha horizontal 
br: > Quebra de linha para separar frases do mesmo parágrafo
address: > forma para endereços dentro de um parágrafo
style: > body { > background-color > } > </style>: Para alterar a cor de 
fundo do site e demais opções no corpo do site através da 
configuração principal <head>.
Negrito com semântica: Tag -> <strong>
Italico com semântica: Tag -> <em>
Para selecionar tags de maneira mais praticas para textos já feitos
CTRL + Shift + P -> selecionar a opção Wrap with abbreviation ou 
envelope com abreviatura.
