marcando com cores, alterando tamanhos e mostrando uma boa organização. Um Clean Code 
-
CTRL + Clique em cima de um arquivo que não exista. Cria uma nova página por exemplo. 
<link rel="stylesheet" href="style.css"> -> 
Utilizado para criar uma aba dentro do site fora do HTML, dessa forma generalizando os estilos dentro do site. 
Apos abrir a CSS externa para estilos, criar uma regra "@charset "UTF-8";" para conseguir utilizar acentuações dentro da página "style.css".
Toda regra inicia-se com "@" para formalizar dentro do código. 
<style="text-align: right;"> -> tag utilizada para mover algo para direita.

Harmonia de Cores/Círculo Cromático:
Cores primárias: amarelo, azul e vermelho
Cores secundarias: laranja, violeta e o verde
Cores terciárias: amarelo-esverdeado, amarelo-alaranjado, vermelho-alaranjado, 
vermelho-arroxeado, azul-arroxeado, azul-esverdeado.
Total de 12 cores no circulo cromático.
Cores frias: lado azul e verde
Cores quentes: lado amarelo e vermelho
-
Cores Complementares: Violeta e amarelo, vermelho e verde, azul e laranja.
Cores Análogas: Cores vizinhas
Cores Análogas + Complementares: Cores vizinhas + cores acima.
Cores Análogas Relacionadas: Duas cores vizinhas pulando uma cor ao lado para a terceira cor.
Cores Intercaladas: Pulando de cor em cor ao lado.
Cores Triádicas: Pulando três cores para somar três cores.
Cores em Quadrado: Pulando duas cores para somar quatro cores.
Cores: Tetrádicas: cor complementares se cruzando formando um retângulo 
Monocromia: Escolher uma cor e alterar saturação e brilho; "Degradê"
